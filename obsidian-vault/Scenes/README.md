# Scenes

Drop your exported Director's Lab scene `.md` files here.

Each scene is exported with:
- YAML frontmatter (`title`, `scene_id`, `date`, `tags`, `beat_map`)
- Beat-by-beat breakdown with visual description, dialogue, direction notes
- Embedded image and video links
- Beat map table

## How to Export

1. Generate a scene in [Director's Lab](http://localhost:5173)
2. Click **📓 Obsidian** in the sidebar
3. Move the downloaded `.md` file into this folder
4. Open in Obsidian → it's ready with graph links and frontmatter

## Tip — NotebookLM

Click **🔬 NotebookLM** to copy the scene text and open NotebookLM. Paste as a source to:
- Ask Gemini questions across multiple scenes
- Generate a story bible
- Find recurring themes and character patterns
